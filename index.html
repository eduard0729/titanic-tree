Buna!

Acesta este un proiect de "Decision tree learning" imbinat cu un proiect de acces la informatii prin API (Framework - Falcon)

Datele utilizate pentru decision tree learning - https://www.kaggle.com/c/titanic/data

Codul sursa - https://github.com/eduard0729/titanic-tree

Pentru trimiterea unui profil de pasager de pe titanic se va acces prin request de tip POST - http://titanic.tekit.ro/predict cu urmatorul json in structura:

{"PassengerId": 5
"Pclass": 3,
"Sex": 1 (0 - female, 1 -male),
"Age": 21,
"SibSp": 0,
"Parch": 0,
"Fare": 7.82,
"Embarked": 1
}

Se va obtine prin request tot un json

{
"Status": "Dead"
}

Tehnologii utilizate:

Decision tree learning: Python 2.x - sklearn, pandas, numpy, matplotlib;
API - Python 2.x (Falcon);
Prezentare: - HTML + CSS (Bootstrap);
Server: Ubuntu 16.04.3 x64 (DigitalOcean), Nginx + Gunicorn
